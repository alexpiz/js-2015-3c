JS soporta: Sistema runtime execution basado en datos que representan valores numéricos, booleanos y cadenas. Funciones, que pueden ser propiedades de objetos Un modelo de objetos basado en prototipos, en lugar de uno basado en clases (Java)

¿Qué es javascript? Inventado por NETSCAPE Lenguaje de scripting multiplataforma orientado a objetos No es útil como lenguaje independiente Su núcleo contiene: Contiene 1 set de objetos predefinidos como array, Date y Math Contiene 1 set de elementos como operadores, estructuras de control y declaraciones Complementado por objetos adicionales: “extensiones” del lado cliente que permiten controlar el browser y el DOM, como formularios, eventos del click, etc. “extensiones” del lado del servidor extendiendo el nucleo proporcionando objetos relevantes para el servidor. Permiten a las apps comunicarse con una base de datos, proveyendo un flujo constante de información.
